# RAGE
 
A rage platformer made for a Game Jam. Theme: Rage.
 
## Requirements
 
- Python 3.x
- Pygame 2.x
 
## Setup
 
```bash
pip install pygame
python rage_game.py
```
 
## Controls
 
- Arrow keys or WASD to move
- Space / W / Up to jump (double jump allowed)
- R to retry after death
- Q to return to menu
 
## Levels
 
1. Tutorial... or is it?
2. Getting Spiky
3. Invisible Platforms
4. The Final Rage
 
## Notes
 
- Deaths are tracked and displayed on screen
- The rage meter fills as you die. At 100% your controls invert temporarily.
- Beat all 4 levels to win.
